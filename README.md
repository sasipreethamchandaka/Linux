# 🐧 What is Linux?

Linux is a **free, open-source, UNIX-like operating system** used in servers, cloud infrastructure, DevOps, cybersecurity, networking devices, embedded systems, and even mobile phones (Android).

What makes Linux special?

- Multi-user  
- Multi-tasking  
- Highly secure  
- Reliable and stable  
- Developer-friendly  
- Runs everywhere (servers → supercomputers → IoT)

---

# 🕰️ How Linux Arrived (Short History)

A very short and simple timeline:

| Year | Event |
|------|--------|
| **1969** | UNIX was created at Bell Labs — foundation for Linux concepts. |
| **1983** | GNU Project started by Richard Stallman (goal: free operating system). |
| **1991** | Linus Torvalds created the **Linux kernel** and released it publicly. |
| **1992 onwards** | Linux + GNU tools combined into a full OS (GNU/Linux). |
| **2000s–Now** | Linux becomes dominant in servers, cloud, DevOps, containers, Kubernetes, etc. |

Today, Linux powers:

- 96% of cloud servers  
- 100% of the top supercomputers  
- Most DevOps tools (Docker, Kubernetes, Jenkins, Terraform)  
- Android smartphones
